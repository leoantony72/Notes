1. **%s : string**
2. **%d : int**
3. **%f : float**
4. **%b : Byte**
5. **%x : returns hex**
6. **%v : {1,2} prints only the value of struct**
7. **%+v : {x:1, y:2} prints value with struct field**
8. **%#v : main.Point{x:1 , y:2 }**
9. **%p : pointer address**
10. **%T : type of value**
